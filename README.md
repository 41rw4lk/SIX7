# SIX7 Privacy Policy

Six7 is a Chrome extension that adds temporary private messaging and peer-to-peer file sharing between Six7 users on the same YouTube watch page.

## Data Six7 may use

- The current YouTube watch page/video identifier, used to group users into a temporary room.
- The user's YouTube @handle/display name, avatar, local settings, block list, and room/session state stored in the browser.
- Private messages and file transfers sent between users who choose to chat or share files.

Six7 uses the user's YouTube @handle/display name only as an in-room display identity for other Six7 users. Six7 does not collect real names, email addresses, phone numbers, addresses, payment information, government identifiers, or other real-world identifying information.

## How data is used

Six7 uses this data only to provide its private messaging and peer-to-peer file sharing features, detect eligible YouTube watch pages, show the in-page overlay, discover compatible Six7 peers, and maintain temporary room/session state.

## Storage

Six7 stores local settings and session-related data in the user's browser using Chrome extension storage or local browser storage.

## Transmission

Peer discovery/bootstrap may use Six7 network services to announce that a Six7 user is present in a temporary room and to help peers connect.

Private messages and file transfers are sent directly between peers using peer-to-peer connections. Six7 network services are used only for peer discovery/bootstrap and connection setup, not for transmitting private message contents or file contents.

## Sharing

Six7 does not sell user data. Six7 does not use user data for advertising, credit, lending, or unrelated purposes. Six7 does not share user data with third parties except as necessary to provide the extension's user-facing messaging and peer-to-peer file sharing features.

## Retention and deletion

Local Six7 data remains in the browser until the user resets their identity, clears extension/browser storage, or removes the extension.

Temporary discovery/session data is used only as needed to provide the service.

## Limited Use

Six7's use of user data is limited to providing and improving the extension's stated private messaging and peer-to-peer file sharing functionality.
