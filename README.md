# SIX7
Privacy policy and public support information for the Six7 Chrome extension.

Six7 Privacy Policy

Six7 is a Chrome extension that adds temporary private messaging and simple peer-to-peer file sharing between Six7 users on the same YouTube watch page.

Data Six7 uses:
- The current YouTube watch page/video identifier to group users into a temporary room.
- A local Six7 handle, avatar, settings, and room/session state stored in the browser.
- Private messages and file-transfer data sent directly between peers when users choose to chat or share files.

Six7 does not sell user data.
Six7 does not use user data for advertising, credit, lending, or unrelated purposes.
Six7 does not scrape YouTube live chat messages or collect the live chat participant list.
Six7 does not host transferred files on Six7 servers.

Peer discovery/bootstrap uses Six7 network services to announce that a Six7 user is present in a temporary room and to help peers connect. Private chats and file transfers are intended to occur peer-to-peer.
